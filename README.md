# Fleet Control App

Modern web application for comprehensive vehicle fleet management, driver monitoring, and fuel consumption tracking.

## Features
- Dashboard with real-time KPIs
- Vehicle & Driver registration
- Fuel consumption analysis (SOLRED/CEPSA integration patterns)
- Maintenance management

## Tech Stack
- HTML5 / CSS3 / JavaScript (Vanilla)
- Premium UI with Glassmorphism and modern aesthetics

## Development
To run the app locally, simply open `index.html` in your browser or use a local dev server:
```bash
# Example using serve
npx serve .
```
